Pokémon Maroon
A fan-made Pokémon RPG playable entirely in your browser — no installation required.
How to Play
Desktop (Windows/Mac/Linux)
Download index.html and double-click it. It opens in any browser.
iPad / Mobile
This game is best played via a hosted link. Visit the live site at:
https://2691434.playcode.io
Online Editors
You can also paste the code into Playcode.io and hit Run.
Features

200+ Pokémon with pixel art sprites
Custom starter lines (Leafkick, Fluppy, Tidalkit)
Mega evolutions, legendary encounters, and raid battles
Fusion system, Pokédex, and full battle mechanics
Easter eggs 👀

Notes

Works in Chrome, Safari, Firefox, and Edge
No internet connection required (desktop only)
All progress is saved locally in your browser
